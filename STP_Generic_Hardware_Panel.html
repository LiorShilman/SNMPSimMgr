§1 Introduction
1.1 Purpose

This document defines the Software Test Procedure (STP) for the Generic Hardware Panel system. The system provides a unified web-based management interface for hardware devices that communicate via SNMP (v2c / v3) or IDD (Interface Design Document) protocols. It covers real-time monitoring, configuration (SET), bulk operations, automation, connectivity resilience, a graphical System Architecture Page with live hardware status, and a hierarchical System Tree covering all hardware and MCC/GCC interfaces.
